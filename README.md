# ｽﾀｯｸﾁｬﾝ デブサミ2026 プレゼンテーション

Reveal.js を使用した、デブサミ2026登壇用のスライドプロジェクトです。

## 🚀 クイックスタート

### 開発環境

Node.js (v20以上推奨) がインストールされていることを確認してください。

1. 依存関係のインストール

```bash
npm install
```

2. ローカルサーバの起動

```bash
npm run dev
```

`http://localhost:5173` でプレビューが開きます。

### ビルドとデプロイ

GitHub Pages で公開するための静的ファイルを生成します。

1. ビルド

```bash
npm run build
```

2. 公開dist/ ディレクトリの内容を GitHub の gh-pages ブランチ、または main ブランチの docs/ フォルダに配置してください。

## 🛠 スライドの編集方法

index.html 内の <script type="text/template"> タグ内に Markdown 形式で内容を記述しています。

* スライドの区切り: --- (水平線)
* 垂直スライド（サブスライド）: --
* 画像レイアウト: layout-tile や layout-full クラスを使用可能です。
* グラフ: chart クラスを持つタグで JSON データを記述します。

## 💡 今後の応用例

1. インタラクティブなデモ: スライド内に JavaScript を直接記述できるため、Web シリアル API を使ってブラウザから実際のｽﾀｯｸﾁｬﾝを操作するデモを組み込むことが可能です。
2. リアルタイムフィードバック: Firebase などのバックエンドと連携し、聴衆の反応（いいね等）をスライド上にリアルタイム表示できます。
3. カスタムテーマ: CSS 変数を書き換えるだけで、ｽﾀｯｸﾁｬﾝのブランドカラー（イエロー/ブラック）に合わせた独自のルック＆フィールに変更できます。
